## 📦 Features

- ✅ Responsive product grid layout
- 🔍 Real-time search with debounce
- 🗂️ Category filtering
- 🔃 Price-based sorting
- 🛒 Add/remove items from cart
- ⚠️ Graceful error handling and empty states
